# Code of Conduct

## Our pledge

We aim to make this project useful, respectful, and practical for founders, operators, and contributors.

## Expected behavior

- Use clear and respectful language.
- Focus feedback on the work.
- Keep examples synthetic unless you have permission to share them.
- Respect privacy and confidentiality.

## Unacceptable behavior

- Harassment or discriminatory language
- Publishing private customer or company data
- Misrepresenting synthetic examples as production results
- Adding secrets, API keys, or credentials

## Reporting

Open a GitHub issue or contact the maintainer through the profile linked from this repository.
