# Security

## Supported versions

The current public version is the supported version.

## Data safety

The repository includes only synthetic and fictionalized sample data.

Before using this with real company data:

- Remove private customer details if the repo is public.
- Do not commit secrets or API keys.
- Review generated outputs before sharing.
- Keep sensitive customer context in private systems.

## Reporting a vulnerability

Open a GitHub issue with a clear description. Do not include secrets, private data, or confidential customer information.
