# Feedback Tracker Template

Use this structure for your own CSV. Keep the column names unchanged for the first run.

| Column | Example value |
| --- | --- |
| feedback_id | FB-001 |
| feedback_date | 2026-04-01 |
| source | Sales call |
| source_detail | Discovery recap |
| customer_name | Example Customer |
| segment | Mid-market |
| industry | SaaS |
| account_value | 75000 |
| lifecycle_stage | Sales |
| feedback_type | Product gap |
| feedback_theme | CRM integration |
| feedback_text | Buyer needs native CRM sync before pilot |
| product_area | Integrations |
| requested_feature | Salesforce two-way sync |
| severity | High |
| frequency_signal | Repeated |
| revenue_blocked | 75000 |
| retention_risk | Medium |
| expansion_potential | High |
| strategic_alignment | High |
| customer_urgency | High |
| workaround_available | Partial |
| implementation_effort | Medium |
| confidence_level | High |
| owner | Product |
| current_status | New |
| next_step | Validate integration scope |
| notes | Internal context |

Suggested allowed values:

- Severity: Critical, High, Medium, Low
- Frequency signal: Widespread, Repeated, Some, Single
- Retention risk: Critical, High, Medium, Low, None
- Expansion potential: High, Medium, Low, None
- Strategic alignment: High, Medium, Low, Outside
- Customer urgency: High, Medium, Low
- Workaround available: Yes, Partial, No
- Implementation effort: Low, Medium, High, Very high
- Confidence level: High, Medium, Low
